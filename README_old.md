# Apostrophe

A premium hub for high-quality patched apps, featuring a modern glassmorphic design and one-click downloads.

## Features

- **Curated Selection**: Access trusted builds of YouTube, YouTube Music, Instagram, and more.
- **Modern UI**: A sleek, responsive interface built with glassmorphism aesthetics.
- **Direct Downloads**: Links are automatically updated from reliable open-source releases.
- **Privacy Focused**: Tools designed to enhance your privacy and user experience.

## Usage

Visit the hub to browse and download the latest versions of your favorite patched applications.

## Disclaimer

All code is open-source. This project hosts links to APKs compiled by trusted third-party developers. We do not modify the APKs ourselves.
